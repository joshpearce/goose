<task id="1.2" type="execute">
<title>Criar RemoteServerPersistence.swift (UserDefaults + Keychain)</title>
<read_first>
- GooseSwift/CodexEmbeddedAuth.swift — padrão Keychain a replicar: `SecItemAdd`, `SecItemCopyMatching`, `SecItemDelete` com `kSecClassGenericPassword`, `kSecAttrService`, `kSecAttrAccount`, `kSecAttrAccessibleAfterFirstUnlockThisDeviceOnly`
- GooseSwift/OnboardingPersistence.swift (ou qualquer ficheiro com `OnboardingStorage`) — padrão de chaves UserDefaults dot-namespaced (`static let` em enum)
- GooseSwift/MoreRouteModels.swift — lido em 1.1; confirmar que `.remoteServer` foi adicionado
</read_first>
<action>
Criar `GooseSwift/RemoteServerPersistence.swift` com:

**UserDefaults keys (enum RemoteServerStorage):**
- `static let serverURL = "goose.remote.serverURL"` — `String`, default `""`
- `static let uploadEnabled = "goose.remote.uploadEnabled"` — `Bool`, default `false`

**URL validation (enum RemoteServerURLValidator):**
- `static func validate(_ raw: String) -> Bool` — retorna `true` se:
  - `URL(string: raw)` não é nil
  - `URLComponents(string: raw)?.scheme` é `"http"` ou `"https"`
  - `URLComponents(string: raw)?.host` não é nil, não está vazio, e não é apenas dígitos e pontos (regex: `^[0-9.]+$` negado)

**Keychain (enum RemoteServerKeychain):**
- `private static let service = "goose.remote"`
- `private static let account = "apiKey"`
- `static func saveToken(_ token: String) throws` — usa `kSecClassGenericPassword`, apaga entrada anterior com `SecItemDelete` antes de `SecItemAdd`, atributo `kSecAttrAccessibleAfterFirstUnlockThisDeviceOnly`
- `static func loadToken() throws -> String?` — usa `SecItemCopyMatching` com `kSecReturnData: true`, `kSecMatchLimit: kSecMatchLimitOne`; retorna `nil` se `errSecItemNotFound`
- `static func deleteToken() throws` — usa `SecItemDelete`; ignora `errSecItemNotFound`

**Erros (enum RemoteServerKeychainError: Error, LocalizedError):**
- `case saveFailed(OSStatus)`
- `case deleteFailed(OSStatus)`

Importar `Foundation` e `Security`.
</action>
<acceptance_criteria>

- `RemoteServerStorage.serverURL` == `"goose.remote.serverURL"` (verificável por `grep`)

- `RemoteServerKeychain.service` == `"goose.remote"` (verificável por `grep`)
- `RemoteServerKeychain.account` == `"apiKey"` (verificável por `grep`)
- `RemoteServerURLValidator.validate("https://meu-servidor.local")` retorna `true` — verificável por leitura do código (hostname não é só dígitos)

# Milestone v14.0: Android Port, BLE Reliability & Protocol Depth

**Status:** ✅ SHIPPED 2026-06-21
**Phases:** 98–111
**Total Plans:** 22

## Overview

v14.0 launched the Android port (Kotlin/Compose + JNI bridge to Rust core), closed all known Gen4/Gen5 historical sync bugs, added BLE reliability improvements (MTU 247, LE 2M PHY, off-wrist detection), battery level for all generations, hardened WHOOP MG detection, removed all production `.unwrap()` calls from Rust, added SQLite connection pool, and documented protocol byte offsets + FFI safety contracts.

## Milestone Summary

**Key Accomplishments:**
1. Android port shipped: Kotlin/Compose + JNI bridge to Rust core, BLE stack, historical sync, metrics display, server upload

3. BLE reliability: MTU 247 + LE 2M PHY negotiation; off-wrist detection via cmd 0x54

5. Code health: 0 production `.unwrap()` in Rust; SQLite connection pool (r2d2); bot audit closed (#59)
6. Protocol documentation: byte-offset WHY comments + FFI/JNI SAFETY blocks

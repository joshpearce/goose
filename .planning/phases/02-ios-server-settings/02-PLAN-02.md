<action>
Criar `GooseSwift/MoreRemoteServerViews.swift` com:

**MoreRemoteServerViewModel** (`@MainActor final class MoreRemoteServerViewModel: ObservableObject`):
- `@Published var serverURL: String` — inicializar com `UserDefaults.standard.string(forKey: RemoteServerStorage.serverURL) ?? ""`
- `@Published var bearerToken: String` — inicializar com `(try? RemoteServerKeychain.loadToken()) ?? ""`
- `@Published var uploadEnabled: Bool` — inicializar com `UserDefaults.standard.bool(forKey: RemoteServerStorage.uploadEnabled)`
- `@Published var urlValidationError: String?` — nil quando sem erro
- `@Published var saveSuccess: Bool = false`

- `func save()`:
  1. Guard `RemoteServerURLValidator.validate(serverURL)` — se falhar: `urlValidationError = "URL inválida. Use https://hostname (não IPs numéricos)."` e return
  2. `urlValidationError = nil`
  3. `UserDefaults.standard.set(serverURL, forKey: RemoteServerStorage.serverURL)`
  4. `UserDefaults.standard.set(uploadEnabled, forKey: RemoteServerStorage.uploadEnabled)`
  5. `try? RemoteServerKeychain.saveToken(bearerToken)` (ignorar erro — não bloquear save se token falhar, o erro é silencioso mas o token não é guardado)
  6. `saveSuccess = true`

**MoreRemoteServerView** (`struct MoreRemoteServerView: View`):
- `@StateObject private var vm = MoreRemoteServerViewModel()`
- Layout com `Form`:
  - `Section("Server")`:
    - `TextField("https://meu-servidor.local", text: $vm.serverURL)` — `.keyboardType(.URL)`, `.autocorrectionDisabled()`, `.textInputAutocapitalization(.never)`
    - Se `vm.urlValidationError != nil`: `Text(vm.urlValidationError!)` com `.font(.caption)`, `.foregroundStyle(.red)` — posicionado inline após o TextField dentro da mesma Section
  - `Section("Authentication")`:
    - `SecureField("Bearer token (API key)", text: $vm.bearerToken)` — `.autocorrectionDisabled()`, `.textInputAutocapitalization(.never)`
  - `Section("Upload")`:
    - `Toggle("Enable Upload", isOn: $vm.uploadEnabled)`
  - `Section`:
    - `Button("Save") { vm.save() }` — `.frame(maxWidth: .infinity)`, `.foregroundStyle(.white)` dentro de um `Label` ou apenas `Text`
- `.navigationTitle("Remote Server")`
- `.navigationBarTitleDisplayMode(.inline)`
- `.listStyle(.insetGrouped)`
- `.gooseListBackground()`
- `.alert("Configurações guardadas", isPresented: $vm.saveSuccess) { Button("OK") {} }`
</action>

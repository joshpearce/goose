<objective>
Add an optional `stage_segments: Option<Vec<SleepStageSegment>>` field to `HrvInput` and implement 3-tier slow-wave-sleep (SWS) window selection so overnight RMSSD is computed over the most physiologically representative deep-sleep window: (Tier 1) the last deep-sleep episode >= 5 min; (Tier 2) a recency-weighted mean across all deep episodes; (Tier 3) full-night fallback. Expose `window_tier_used: u8` on `HrvOutput`. Also document the ALG-HRV-04 cross-validation gate (<= 1 ms vs the my-whoop Python reference) as a code comment, since it is a manual/human verification step. (ALG-HRV-03, ALG-HRV-04)

Purpose: WHOOP computes overnight HRV during deep sleep; selecting the right SWS window aligns goose RMSSD with the reference.
Output: `HrvInput.stage_segments`; `select_sws_window` free function; `HrvOutput.window_tier_used`; ALG-HRV-04 doc comment; tests for all three tiers.
</objective>

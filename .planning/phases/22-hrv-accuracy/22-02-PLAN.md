<objective>
Implement a Lipponen-Tarvainen-style ectopic beat filter and apply it per gap-segment (from plan 22-01) before RMSSD is computed. The filter uses a rolling 5-beat local median reference and rejects any interval whose absolute deviation from the median exceeds 0.20 × median. Expose `ectopic_filter_removal_fraction: f64` on `HrvOutput` so the removal rate is observable (and consumable by the Recovery V2 dashboard). (ALG-HRV-02)

Purpose: Ectopic beats produce spurious large successive differences that the 300–2000 ms range gate misses; the median-relative filter removes them before RMSSD.
Output: `lipponen_tarvainen_filter` free function; per-segment application in `goose_hrv_v0`; `HrvOutput.ectopic_filter_removal_fraction`; tests proving rejection and a 0.0 baseline.
</objective>

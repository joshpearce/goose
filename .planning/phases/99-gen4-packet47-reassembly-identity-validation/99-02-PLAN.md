<objective>
Implement HISTORICAL_DATA_RESULT identity validation (SYNC-11, issue #163).

Root cause: the cmd 23 (HISTORICAL_DATA_RESULT) handler in `handleHistoricalCommandResponse`
reads the result code and immediately calls `completeHistoricalSync` or schedules idle
completion — it never reads the 8-byte hardware identity the strap echoes back at
`payload[5..<13]`. A response from a different device or a spoofed ACK would silently complete
the sync with the wrong device's data.

Fix:
1. Add `var connectedStrapIdentity: [UInt8]? = nil` to `GooseBLEHistoricalManager`.
2. In the `getDataRange` success branch, capture `Array(payload[5..<13])` when `payload.count >= 13`
   (the Gen4 path already guards `payload.count >= 14`, so this is covered) and store it to
   `historicalManager.connectedStrapIdentity`. Also log the raw hex for first-run hardware verification.
3. In the `historicalDataResult` branch, before the existing complete/schedule logic, guard
   `payload.count >= 13`; extract `Array(payload[5..<13])`; if `connectedStrapIdentity` is set
   and bytes differ, call `failHistoricalSync` with the identity_mismatch reason and return.
   If identity is not yet set, log `identity_check_skipped` and continue.
4. Clear `connectedStrapIdentity` in sync begin, `completeHistoricalSync`, and `failHistoricalSync`.

Purpose: Ensure historical sync data is only accepted from the device that initiated the session.
Output: Two modified Swift files; no Rust changes; issue #163 closed.
</objective>

        case .historicalDataResult:

    Replace with (insert identity validation before the existing logic):

        case .historicalDataResult:
          // SYNC-11: validate 8-byte strap identity echoed in cmd 23 response
          if payload.count >= 13 {
            let respondingIdentity = Array(payload[5..<13])
            if let expectedIdentity = historicalManager.connectedStrapIdentity {
              if respondingIdentity != expectedIdentity {
                let expectedHex = Data(expectedIdentity).hexString
                let gotHex = Data(respondingIdentity).hexString
                record(
                  level: .error,
                  source: "ble.sync",
                  title: "historical_sync.identity_mismatch",
                  body: "expected=\(expectedHex) got=\(gotHex)"
                )
                failHistoricalSync("HISTORICAL_DATA_RESULT identity mismatch: expected \(expectedHex), got \(gotHex).")
                return
              }

                level: .warn,
                source: "ble.sync",
                title: "historical_sync.identity_check_skipped",
                body: "connectedStrapIdentity not set; skipping cmd 23 identity check"
              )
            }
          }
          historicalManager.pendingHistoryEndAckPayload = nil
          if historicalManager.historyCompleteReceived {
            completeHistoricalSync(reason: "history_complete")
          } else {
            scheduleHistoricalIdleCompletion(reason: "history_end_ack_idle")
          }

<objective>
Phase 122 delivers two features on top of Phase 119's completed stealth data layer:

1. Health dashboard "—" gate — HealthMetricSnapshot gains stealthKey: String = "" and displayValue returns an em dash (U+2014) for any of the 6 targeted metrics when the user has hidden it via GooseStealthMode.isHidden(metric:). The gate is at the model layer, not the view layer, because all metric cards render generically via snapshot.displayValue. All 8 are addressed in this revised plan.

Purpose: Users can hide individual sensitive metrics from the Health dashboard without losing the underlying data.
